This shirt is one project I'm most proud of mainly because every aspect of this project was done by scratch. To begin this project, I first carved out a stamp with a moth design. The stamp was then pressed in ink and pressed into my sketchbook until the desired result. The graphic was then imported to Illustrator and made into a vector image. From here, a repeat pattern was made and imported to Spoonflower, a patterned fabric online textile service. After getting the fabric, I created a square neck top, pictured here.

To finish up this project, I decided to make it like a garment you would buy in a store. I created an embroidered tag using a coding program called Pembroider, which utilizes vector graphics to make the coding process easy. The embroidered tag is featured below along with some detail images.
